# SpineViewer


![4.2](misc/screenshot.png)

## How to code

```
#macos
premake5 --file=premake5.lua xcode4
open build/SpineViewerProject.xcworkspace
```

Base on:

1. [Spine Runtime 2.x - 4.x](https://github.com/EsotericSoftware/spine-runtimes/)
2. [GLFW](https://www.glfw.org/)
3. [OpenGL](https://www.opengl.org/)
4. [ImGui](https://github.com/ocornut/imgui)
